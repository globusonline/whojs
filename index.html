Philosophy

WhoJS is a library aimed at making it simpler and more expressive for Front End Developers to test the displays fo their web page. Using WhoJS allows you to emulate user navigation within a single page using JavaScript, in a simple, easy to understand syntax. It's easy to inspect the DOM for elements, but we want to make sure Who is seeing exactly what the user is seeing.

Getting Started

Grab yourself a copy of WhoJS from GitHub and include it in your page similar to this:


<script src="javascripts/who.js"> </script>

Using WhoJS

Setting Context

Setting context is useful for zoning in on a specific part of the page Who is looking at. Running this example will quickly show you a shift in Who's context. The part of the page that Who is actively looking is outlined in blue. You must set this style for yourself, by overriding the CSS class .who-context.


Who.setContext('#new_context_example');

Reset Context

Sometimes you want Who to be watching over the whole page. A quick reset of context can be achieved using resetContext(). You'll notice when you run this code that the context is shifted back to the body.


Who.resetContext();

Typing Values Into a Form

When a Who user wants to type a value into a form, there are two things they need. The text of the label they are looking for, and the text, which is the string that will be entered into the field.


Who.setContext('#typing_values_example');

Who.typesValueIntoForm({
	label : "Name",
	text : "Neil Dahlke"
});

By default, the labels are case insensitive. This can be overriden by setting the caseSensitive option to true and can also be search by partial matches using the findByPartial flag.


Who.setContext('#typing_values_example_2');

Who.typesValueIntoForm({
	label : "tail",
	text : "Shark Fin",
	findByPartial : true
});

// This will fail as there is no findByPartial flag

Who.typesValueIntoForm({
	label : "tail",
	text : "Bottle Neck"
});

Typing the Enter Key


Who.setContext('#typing_enter_example');

Who.typesValueIntoForm({
	label : "Alley",
	text : "Oop",
	findByPartial : true
});

Who.typesEnterKey({
	label : "Alley",
	findByPartial : true
});

Clicking on elements


Who.setContext('#click_example');

Who.clicksElement({
	text : "Click Me!",
	type : "a"
});

or you can specify links using the LINK keyword.


Who.setContext('#click_example');

Who.clicksElement({
	text : "Click",
	type : "LINK",
	findByPartial : true
});
Click Me!

Shortcuts for clicking on buttons and links.

When you are writing several Who tests at a time, you may want to have some shortcuts. For clicking on frequently clicked elements, like buttons and anchors, Who provides clicksButton and clicksLink.


Who.setContext('#click_shortcut_example');

Who.clicksButton({text : "Click Me!"});

You can also use the same syntax for clicking on an anchor.


Who.setContext('#click_shortcut_example');

Who.clicksLink({text : "Click Me Too!"});

It even works with checkboxes and radio buttons.


Who.setContext('#click_shortcut_example');

Who.clicksCheckbox({label : "Me too?"});

Who.clicksRadio({label : "No", findByPartial : true});
Click Me Too!

Reading form values

Often times you'll simply want Who to read in the form values, and for that, we have readsFormValue. In the example below, who will enter in a name, and then read it back to us.


Who.setContext('#read_values_example');

Who.typesValueIntoForm({
	label : "my name",
	text : "J Dilla",
	findByPartial : true
});

var formVal = Who.readsFormValue({
	label : "My NaMe Is:"
});

alert('Hi '+(formVal||"Who")+'!');

Passing time

When you need Who to chill for a moment, maybe while you're fetching some things with AJAX, you can use passesTime with the option thenWho as a required callback for when Who finishes waiting.


Who.setContext('#passing_time_example');

var _start = new Date();

Who.passesTime({
	ms : 2000,
	thenWho : function() {
		var _end = new Date(),
			_waited = ((_end - _start)/1000);
		alert("Who waited "+_waited+" seconds!");
	}
});

Hovers Over


Who.setContext('#hovers_over_example');

Who.hoversOver({
	text : "SICK!",
	type : "div",
	msToOut : 1000
});

Sees an Element with Text


Who.setContext('#sees_element_with_text_example');

var _seesOurGreeting = Who.seesElementWithText({
			    "text":"prosper",
			    "type":"span",
			    findByPartial : true
			});

_seesOurGreeting == true ? alert('Good to see you!') : null;
Live Long and Prosper!

Sees an Element with Placeholder Text


Who.setContext('#sees_element_with_placeholder_text_example');

var _seesFavoriteFootballTeam = Who.seesElWithPlaceholderText({
				    "placeholderText":"Chicago Bears"
				});

_seesFavoriteFootballTeam == true ? alert('Now you know.') : null;

Seeing Images


Who.setContext('#sees_image_example');

var _seesImage = Who.seesImages();

_seesImage == true ? alert('"I see", said the blind man.') : null;
